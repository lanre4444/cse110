# CSE 110 – Introduction to Programming

This repository contains my coursework, assignments, and projects completed for **CSE 110: Introduction to Programming**, with a focus on developing foundational programming skills using **Python**.

## Course Topics

* Variables and data types
* Conditional statements
* Loops and iteration
* Functions
* Lists and strings
* File handling
* Data analysis
* Problem-solving and algorithm design

## Projects Completed

* **Clever Stories** – A Mad Libs–style program that generates creative stories from user input.
* **Meal Price Calculator** – Calculates meal costs, sales tax, tips, and the final total.
* **Adventure Game** – A text-based interactive game where users make choices that affect the outcome.
* **Word Puzzle** – A word-guessing game that challenges players to identify a hidden word through clues or guesses.
* **Shopping Cart** – A console-based application for adding items, viewing a cart, calculating totals, and managing purchases.
* **Data Analysis** – Processes and analyzes datasets to compute statistics and extract meaningful insights using Python.

This repository reflects my progress in learning programming concepts and applying them through practical projects and assignments.
